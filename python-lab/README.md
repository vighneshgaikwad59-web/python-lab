# python-lab
This is my first Python project on GitHub.

The project calculates a person's **Body Mass Index (BMI)** using their weight and height, then displays the BMI value along with the corresponding health category.

## 📌 Features

- Accepts user weight (kg) and height (cm)
- Converts height from centimeters to meters
- Calculates BMI using the standard formula
- Displays BMI rounded to 2 decimal places
- Classifies BMI into:
  - Underweight
  - Normal Weight
  - Overweight
  - Obese
- Includes a reusable `calculate_bmi()` function
